# Onyx Wallet Onboarding Process

This document describes the functional steps an Onyx Wallet user goes through during the initial setup.

## 1. Introduction Screens (Info)
The onboarding begins with a series of high-level introductory slides highlighting the core values of the application:
- **Welcome to ONYX WALLET**: An introduction to the ultimate wealth management interface.
- **Absolute PRIVACY**: Explaining that data is stored locally and securely, with no cloud tracking.

Users can navigate through these slides or skip directly to the setup.

## 2. Identity Establishment (Identity)
The user is asked to create their local profile:
- **Identity Name**: The name used for the profile.
- **Secure Passcode**: A key required to access the application, reinforcing the focus on privacy and security.

## 3. Financial Profiling Quiz (Quiz)
A 5-question assessment designed to determine the user's "Strategic Path". This helps tailor the experience to their financial personality.
- **Questions include**: Primary financial focus, budgeting style, investment preferences, risk tolerance during market downturns, and overall philosophy on wealth.
- **Options lead to different paths**: Investor, Frugal, Guardian, Legacy, Catalyst, Nomad, Alchemist.

## 4. Strategic Alignment Result (Path Result)
Based on the quiz answers, the app reveals the user's recommended "Financial Path":
- **Examples**: The Strategic Investor, The Frugal Architect, Wealth Guardian, etc.
- A description of the path's focus (e.g., "Master of discipline" for Frugal Architect).
- The user has the option to retake the assessment if they feel the result doesn't match their goals.

## 5. First Deposit & Wallet Initialization (Wallet)
The user sets up their primary financial reserve account:
- **Wallet Type**: Selection between Bank Account, Credit Card, Cash, Mobile Money, or Crypto.
- **Currency**: Choice from a wide range of supported global currencies through a searchable selector.
- **Wallet Label**: A custom name for the wallet (e.g., "Main Savings").
- **Initial Balance**:
    - For standard wallets: Entry of the current balance.
    - For Crypto wallets: Entry of the quantity and current price to calculate the total value automatically.

## 6. Finalization (Finish)
A confirmation screen ("Vault Ready") indicating that the profile is secured and the initial reserve is initialized.
- Clicking the **"Launch Dashboard"** button completes the onboarding, saves the profile and wallet to the local database, and takes the user to the main app dashboard.
