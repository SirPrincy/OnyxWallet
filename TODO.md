# Project Roadmap & Improvement Ideas

This document outlines the planned rework for the "Evolving Paths" system and provides a list of strategic ideas for future enhancements.

## 1. Rework: Evolving Paths (Level-Based Titles)
Modify the `WealthPath` system to include dynamic titles that evolve as the user's Level/Tier increases.

### Concept: Title Tiers
- **Level 1-10 (Bronze/Silver)**: Entry-level, slightly humorous or "struggling" titles.
- **Level 11-25 (Gold)**: Developing, more serious and professional titles.
- **Level 26-49 (Platinum/Diamond)**: Elite, mastery-level and institutional titles.
- **Level 50-75 (Archon/Emerald)**: Legendary, high-stake leadership titles.
- **Level 76-100 (Mythic)**: Onyx Legend, end-game financial sovereignty.

### Examples for "The Investor"
- **Level 1**: Investisseur Fauché
- **Level 5**: Investisseur du Dimanche
- **Level 10**: Investisseur Débutant
- **Level 20**: Investisseur Pro
- **Level 50**: Hedge Fund Investisseur
- **Level 100**: Onyx Legend (End-game Mastery)

### Examples for "The Frugal Architect"
- **Level 1**: Chasseur de Promos (Promo Hunter)
- **Level 3**: Économe Prudent (Prudent Saver)
- **Level 5**: Maître du Budget (Budget Master)
- **Level 8**: Architecte de l'Abondance (Architect of Abundance)

## 2. 12+ Strategic Improvement Ideas

1.  **Biometric Bypass**: Allow users to replace the manual passcode with FaceID/Fingerprint for faster entry.
2.  **Vault Export/Import**: Implement the logic to generate a JSON file of all data for backups.
3.  **Recurring Transaction Automation**: Automatically suggest adding recurring transactions when the same expense is detected twice.
4.  **Multi-Currency Support for Wallets**: Allow a single wallet to hold balances in different currencies (essential for Crypto).
5.  **Smart Categories**: Use the AI Service to automatically categorize transactions based on their title.
6.  **Milestone Celebration**: Add a "Confetti" effect and a dedicated screen when a Tier or a Savings Goal is reached.
7.  **Emergency Fund Calculator**: A tool that calculates exactly how much the user needs based on their last 3 months of expenses.
8.  **Inflation Visualizer**: Show the "True Value" of cash over time within the savings goals to encourage investing.
9.  **Dark/Light Mode Sync**: Automatically switch the app theme based on the device's system settings.
10. **Budget "Carry Over"**: Option to move remaining budget from one month to the next or automatically put it into a goal.
11. **Path-Specific Quests**: Missions that change depending on your Path (e.g., Frugal path gets "No-Spend Weekend" missions).
12. **Financial News Aggregator**: A dedicated tab for news related to the user's specific investments (Crypto/Stocks).
13. **Anonymous Sync**: Optional end-to-end encrypted sync between devices using a private key (no account needed).
